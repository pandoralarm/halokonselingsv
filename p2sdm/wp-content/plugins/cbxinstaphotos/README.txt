Instagram photo display for wordpress.

== Description ==

This plugin shows photos from instagram account

[Learn More and documentation](https://codeboxr.com/product/cbx-insta-photo-for-wordpress/)
For any help/support please [contact us](https://codeboxr.com/contact-us/)

**Key Features**

* Secured Insta connection through api on provided by client id and secret id
* Layout system, default layout include with core plugin
* Layout override from theme
* Extensive settings(globally & each post) that interacts with different sort of output
* Advanced Native Caching through wordpress transient api
* Responsive & interactive design
* Shortcode
* Classic Widget
* Elementor widget (New in V1.0.5)
* Gutenberg block (New in V1.0.5)

== Installation ==

1. Upload `cbxinstaphotos` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You'll now see a menu called "CBX Insta Photos" in left menu of word, start from there
